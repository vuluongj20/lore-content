# Lore Content

Lore Content is a content management system (CMS) for [Lore](https://github.com/vuluongj20/lore).

## Components
The CMS has three components:
* Front end (Angular)
* Back end (Express.js)
* Login page ([Sign](https://github.com/vuluongj20/sign))

## Try it out
